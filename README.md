# Install

See [Jekyll requirements](http://jekyllrb.com/docs/installation/) and install project dependencies:

````
   $ gem install jekyll bundler
   $ bundle install
````
To run server:
````   
   $ bundle exec rake serve
````
To run tests: 
````   
   $ bundle exec rake test
````
